📤 Upload de artes — Bentô

Suba fotos/artes em resolução cheia (sem limite do chat) e copie os links públicos.
⚙️ Falta configurar o Supabase (uma vez só):
1. Crie um projeto gratuito em supabase.com.
2. Em Storage → New bucket, crie o bucket artes marcando Public bucket.
3. Em Settings → API, copie a Project URL e a chave service_role.
4. Na Vercel → Settings → Environment Variables do site, adicione:
  SUPABASE_URL = Project URL · SUPABASE_SERVICE_KEY = service_role
5. Faça Redeploy e recarregue esta página. Pronto para sempre. ✅
